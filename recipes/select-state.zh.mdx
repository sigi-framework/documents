# StateSelector
在多个组件间共享 **EffectModule** 时，通常会需要从整个 **AppState** 上选择一部分出来给组件使用从而避免无效渲染。
**Sigi** 在 `useEffectModule` 和 `useEffectState` hook 中提供了 **selector** API 来实现这个需求。组件可以从 **EffectModule** 对应的 **AppState** 上选择自己关心的那部分状态，**AppState** 上其它部分的更新会被该组件忽略掉，也不会引起该组件的渲染。
